# 仕様書：『タイミング勝負の猫スタッキング・バカゲー（仮）』

## 【ゲームタイトル】
**ニャンコ・ダイブ！ ～全宇宙ネコ積み上げ計画～**

---

## 【概要】
HTML5 Canvasとシンプルな矩形判定のみを用いた、物理演算なしのタイミング系スタッキング・バカゲー。
左右に往復移動する「猫」を、プレイヤーのタイミング（画面タップ/クリック）で真下に落下させ、どこまで高く積み上げられるかを競います。
真ん中からズレて着地した時の「ガバガバな宇宙ダイブ演出」とシュールな世界観により、1日での爆速実装と高いリプレイ性・中毒性を両立させます。

---

## 【コア体験/ゲーム性】
1. **シンプル＆シビアな目押しアクション**
   - 上部を左右に揺れる猫を、狙ったタイミングで真下に落とすだけ。
2. **ガバガバ判定が生むシュールな爽快感**
   - 土台から少しでもズレると、物理演算を無視して画面外や宇宙へ勢いよくすっ飛んでいく（ダイブ）演出。
3. **ハイテンポなループ**
   - ゲームオーバーからリスタートまで1秒以内。テンポよく何回も遊んでしまう中毒性。

---

## 【操作方法】
- **PC**: マウス左クリック / スペースキー
- **スマホ**: 画面タップ
- **アクション**: 
  - プレイ中：上の猫を真下に落下させる
  - ゲームオーバー時：画面タップで即リトライ

---

## 【開発手順（エンジニアAIへの指示用）】

以下の仕様に従い、単一の `index.html` ファイル（HTML + CSS + 簡易JavaScript）として、コピー＆ペーストですぐにブラウザ動作するコードを生成してください。

### 1. 画面構成・UI
- 画面サイズ：縦長（スマホの比率を想定。例：幅400px、高さ600px）のCanvasを画面中央に配置。
- 背景：夜空（黒〜紺色のグラデーション）に、簡単な星のドット。
- UI表示：
  - 現在のスコア（積んだ猫の数）を画面上部に大きく常時表示。
  - ゲームオーバー時に「GAME OVER」テキストと、リトライを促すテキストを表示。

### 2. ゲームループ・状態管理
状態は以下の3つを持つ。
1. `TITLE`: スタート画面（「タップしてスタート」）
2. `PLAYING`: 猫を動かして積んでいる状態
3. `GAMEOVER`: 猫を落とし損ねてゲームオーバーになった状態

### 3. オブジェクト仕様
- **移動中の猫（上部）**:
  - 画面上部（Y=50固定）を、左右に往復移動させる。
  - 速度はスコアが上がるにつれて少しずつ早くする。
- **積み上がった猫（下部〜）**:
  - 一番下の土台（固定：幅200px、高さ30px、画面中央）からスタート。
  - 落下した猫は、その位置に固定され、次の土台（または前の猫）になる。

### 4. 判定と「ガバガバ演出」のロジック
- プレイヤーがタップすると、移動中の猫のX座標を保持したまま真下に落下させる。
- **着地判定**:
  - 直下の猫（または土台）のX範囲内（`prev_x - width < current_x < prev_x + width`）に着地できたら**成功**。
    - スコア加算（+1）。
    - 積み上げられた猫リストに新しい猫を追加。
  - 範囲外（ズレている）の場合、または半分以上はみ出た場合は**失敗（ゲームオーバー）**。
    - **ガバガバ宇宙ダイブ演出**: 失敗した猫は、着地せずにそのままの勢いで画面外（斜め下や宇宙の彼方）へ猛スピードで飛んでいくアニメーション（X/Y座標に毎フレーム加算）を1秒ほど再生し、その後 `GAMEOVER` 状態へ移行する。

### 5. アセット・グラフィック
- 画像ファイルは一切使用しない。
- すべて `Canvas API` の図形描画（`ctx.fillRect`, `ctx.arc` など）で代用する。
- 猫の見た目：
  - 四角い胴体に、簡単な耳（三角形）と目・ヒゲを描画したシュールで可愛い簡易キャラクター（単色＋パーツ）。

### 6. 実装の制約
- 外部ライブラリ（PhaserやMatter.jsなど）は使用禁止。
- 外部画像・音声ファイルも原則なし（完全なシングルファイルHTMLで完結させること）。
