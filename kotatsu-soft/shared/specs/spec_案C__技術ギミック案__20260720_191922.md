# ゲーム仕様書：『Mikan Balancer（仮）』

## 1. ゲームタイトル
**Mikan Balancer（ミカン・バランサー）**

## 2. 概要
本作は、スマートフォンのジャイロセンサー（傾き検知）を直感的に利用し、不安定な足場の上でミカンを落とさないようにバランスを保ち続ける、物理演算ベースの超高難易度アクションゲームです。プレイヤーはスマホ本体を前後左右に傾けることで、画面内の「ミカンが乗った皿」の角度を制御します。

## 3. コア体験 / ゲーム性
*   **コア体験:** 「物理演算による予測不能な挙動」と「プレイヤーの身体感覚」を同期させる没入感。
*   **ゲーム性:**
    *   **ジャイロ操作:** スマホの傾きがそのままゲーム内の皿の傾きに直結。繊細な手ぶれすらもミカンの挙動に影響する。
    *   **物理演算の不安定さ:** ミカンは球体であり、転がりやすい性質を持つ。足場（皿）の摩擦係数や反発係数を調整し、少しの油断でミカンが落下する緊張感を演出。
    *   **スコアアタック:** ミカンを落とさずに耐え続けた時間がスコアとなる。
    *   **環境変化:** ステージが進むごとに、足場の揺れや風などの外部要因が加わり、難易度が上昇する。

## 4. 操作方法
*   **基本操作:** スマートフォン本体を傾ける（ジャイロセンサー）。
    *   右に傾ける → 皿が右に傾く
    *   左に傾ける → 皿が左に傾く
    *   手前に傾ける → 皿が手前に傾く
    *   奥に傾ける → 皿が奥に傾く
*   **リスタート:** 画面タップ（ミカン落下後、即座に再開可能）。

## 5. 開発手順（エンジニアAIへの指示用）

エンジニアAIに対し、以下のステップで実装を依頼してください。

### Step 1: プロトタイプ環境の構築
*   Unity等のゲームエンジンを使用し、3D空間上に「皿（Plane/Mesh）」と「ミカン（Sphere）」を配置する。
*   ミカンにRigidbodyコンポーネントを付与し、物理演算を有効にする。

### Step 2: ジャイロセンサーの実装
*   `Input.gyro` を利用し、端末の傾き情報を取得する。
*   取得した傾き値を、皿オブジェクトの `transform.rotation` にリアルタイムで反映させるスクリプトを作成する。
*   ※感度調整用の変数（Sensitivity）を外部から変更できるように設計すること。

### Step 3: 物理挙動のチューニング
*   ミカンの「摩擦（Friction）」と「反発（Bounciness）」を調整し、転がりやすさを最適化する。
*   皿の端に「落下判定（Trigger）」を設置し、ミカンが皿から離れた瞬間にゲームオーバー処理を呼び出す。

### Step 4: UIとゲームループの実装
*   経過時間を計測するタイマーUIの実装。
*   ゲームオーバー時に「リトライボタン」を表示し、シーンをリロードする機能を実装する。

### Step 5: 演出の追加
*   ミカンが転がる際のSE（転がり音）と、落下時のSEを追加。
*   背景にシンプルな環境音を配置し、没入感を高める。
