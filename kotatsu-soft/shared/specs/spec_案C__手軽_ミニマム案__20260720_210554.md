# 仕様書：『深夜のミカン・ランダム生成』

## 1. ゲームタイトル
**Midnight Orange Generator (深夜のミカン)**

## 2. 概要
ユーザーが画面をクリック（またはタップ）するたびに、CSSの`filter`プロパティを動的に操作し、色味や形状が微妙に異なる「ミカン」をランダム生成するミニマムなWebコンテンツ。深夜（22:00〜04:00）にアクセスすると、通常とは異なる「レアなミカン」が出現する隠し要素を実装する。

## 3. コア体験 / ゲーム性
*   **手軽さ:** 複雑なゲーム性は排除し、クリックするだけの直感的なインタラクションに特化。
*   **偶発性:** CSSフィルターによる色相回転（hue-rotate）や変形（scale/skew）で、毎回異なるミカンが生成される楽しさ。
*   **深夜の特別感:** 特定の時間帯のみ、CSSの`filter`値に特殊なパラメータを適用し、ユーザーに「深夜の発見」という体験を提供する。

## 4. 操作方法
*   **クリック/タップ:** 画面上のミカンをクリックするたびに、新しいミカンが生成される。
*   **自動生成:** ページ読み込み時に1つ目のミカンが表示される。

## 5. 開発手順（エンジニアAIへの指示用）

以下のステップで実装を進めてください。

### Step 1: HTML/CSSの準備
*   `div`要素でミカンの基本形状（円形、オレンジ色）を作成。
*   CSSの`filter`プロパティ（`hue-rotate`, `brightness`, `contrast`）を適用できるようにしておく。

### Step 2: JSによるランダム生成ロジック
*   クリックイベントリスナーを設置。
*   `Math.random()`を使用して、以下の値をランダムに生成し、CSSの`style`属性に適用する。
    *   `hue-rotate`: 0deg 〜 360deg
    *   `transform`: `scale` (0.8 〜 1.2), `skew` (-10deg 〜 10deg)

### Step 3: 深夜限定ロジックの実装
*   `new Date().getHours()`を取得。
*   22時〜4時の間であれば、`filter`に`invert(1)`や`sepia(1)`などの特殊な値を強制的に追加する条件分岐を記述する。

### Step 4: 最適化
*   外部ライブラリは使用せず、Vanilla JS（純粋なJavaScript）のみで記述すること。
*   コード量は最小限（50行以内）に抑えること。
